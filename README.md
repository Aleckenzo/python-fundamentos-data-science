# 🐍 Fundamentos de Python para Data Science

Este repositório contém meus estudos iniciais em Python, desenvolvidos no Google Colab, com foco na construção de base para Análise de Dados.

---

## 📚 Conteúdos Trabalhados

* Variáveis e tipos de dados (`int`, `float`, `str`, `bool`)
* Estruturas condicionais (`if`, `elif`, `else`)
* Estruturas de repetição (`for`, `while`)
* Funções e métodos
* Manipulação de strings
* Listas, tuplas e dicionários
* Entrada e tratamento de dados (`input`, conversão de tipos)

---

## 🛠 Tecnologias

* Python
* Google Colab

---

## 🎯 Objetivo

Desenvolver uma base sólida em Python para aplicação em análise de dados.
